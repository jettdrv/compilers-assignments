Loop * L1 )
static

Funzione che verifica l'adiacenza tra due loop.

Se i loop sono guarded, il successore non loop del guard branch del primo loop deve essere l'entry block dell'altro. Questo si verifica ottenendo i blocchi successori del guard branch e controllando se fanno parte del loop. Appena troviamo il successore che non appartiene il loop, verifichiamo se è uguale al preheader del secondo loop. Se i loop sono unguarded l'exit block del primo loop deve essere il preheader del secondo loop.

Parametri
L1
L2
Restituisce
true se sono adiacenti, false altrimenti

◆ isFusionCandidate()

bool anonymous_namespace{LoopFusion.cpp}::isFusionCandidate ( Loop * L)
static

Funzione che controlla se un loop è adatto alla fusione.

Per essere fuso, un loop deve essere in forma normale, deve possedere tutti i blocchi importanti come il preheader, header, latch ed exiting blocks. Inoltre non deve lanciare eccezioni e non deve contenere accessi volatili sulla memoria.

Parametri
LLoop da controllare
Restituisce
true se il loop L è candidato per la fusione, false altrimenti
